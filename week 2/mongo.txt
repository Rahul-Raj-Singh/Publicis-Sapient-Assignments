Steps:

1. Starting Main(Primary) server at port 27017
	mongod --config C:\Program Files\MongoDB\Server\4.2\bin\mongod.cfg

2. Opening mongoshell
	mongo --port 27017

3. Initiating replica set
	rs.initiate()

4. Starting other 2 servers at 27018 and 27019
	mongod --replSet replica --logpath "2.log" --dbpath rs2 --port 27018
	mongod --replSet replica --logpath "3.log" --dbpath rs2 --port 27019

5. Adding these 2 replica instances in the replica set
	rs.add("localhost:27018");
	rs.add("localhost:27019");

6. Introducing lag in one of the secondary server
	change = rs.conf()
	change.members[2].slaveDelay = 3600; // 1hr=3600s delay to server running on port 27019
	rs.reconfig(change)

7. To monitor status
	rs.status()
